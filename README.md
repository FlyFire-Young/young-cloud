# spring-cloud-eurake #
基于spring-cloud-eurake注册中心。
## 前言 ##
spring-cloud-eurake服务注册中心demo。

## 目录结构介绍 ##

	|-- eurakeclient                            // 客户端
	|-- eurakeclient2                           // 客户端
	|-- eurakeserver                            // 服务端


## 安装步骤 ##

	git clone git@github.com:FlyFire-Young/young-cloud.git      // 把模板下载到本地
  使用eclipse,idea等编译工具导入项目
